## 使い方

Oculus Rift と Leap Motion のテストをするとき適当なシーンがあると便利ですよね。<br>
というわけで適当にわらわらするSDユニティちゃんを見て触れるサンプルプロジェクトです。

SDUnityChanWarawara のフォルダに Main.scene というシーンが入っているので開いて実行してみてください。
